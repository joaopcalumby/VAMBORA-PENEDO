# VAMBORA PENEDO
## O que é o projeto:
Aplicação para centralizar e otimizar o acesso a informações sobre transporte público intermodal (ônibus, vans e balsas) no município de Penedo - AL. - MVP focado em acessibilidade, fornecimento de horários, rotas e canais de feedback para a comunidade local.
